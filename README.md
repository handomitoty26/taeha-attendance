# 태하관자습실출석부v1_1

기숙사 태하관 자습실 출석부 관리 PWA 프로그램

## 파일 구조

```
태하관자습실출석부v1_1/
├── index.html       ← 메인 프로그램
├── manifest.json    ← PWA 매니페스트
├── sw.js            ← 서비스워커 (오프라인 지원)
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## GitHub Pages 배포 방법

1. 이 저장소를 GitHub에 업로드
2. Settings → Pages → Branch: main, Folder: / (root) 선택
3. 저장 후 생성된 URL로 접속
4. 브라우저/탭 상단 "설치" 버튼 또는 메뉴에서 앱 설치

## 사용 환경

- Android Galaxy Tab S9 FE+
- PC 브라우저 (Chrome 권장)
